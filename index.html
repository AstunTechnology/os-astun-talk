A brief intro into open source at Astun, and how we use it to get best value for our clients

Community

We don't just consume open source software, we actively work to improve it and to nurture the community. This is a virtuous circle- we make the software better, and provide better software for our clients

Contributions

We have many core committers and committee members for the PSCs (eg Geoserver, GeoNetwork)

Enterprise Open Source

Our belief is that open source is viable for Enterprise computing (not just GIS) and that we have the skillset and experience with scaling and security to make it a viable choice in any situation
